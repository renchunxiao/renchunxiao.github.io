Request 作用

Request 是对将要运行的测试的抽象描述。如果我们要支持过滤和排序，我们需要一个更抽象的说明，比测试本身要更加丰富。

当运行单元测试的时候，每一个 Runner 都被 Requst 创建。Runner 会返回一个详情 Description 一个树形结构来描述将要运行的测试。
