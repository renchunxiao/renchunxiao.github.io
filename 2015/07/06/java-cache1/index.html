缓存有哪些属性

一般缓存会有如下的属性：

  • 命中率：是查询命中缓存的比率
  • 最大元素：缓存中可以存放的最大元素个数，因为缓存不可能无限大
  • 清空策略：当缓存空间不够时候的清空策略
