RegularEnumSet 里面就是用了一个 long 型的 elements 来存放，因为 long 是 8 个字节，一共 64 bit。并且每个枚举的元素都有 ordinal 属性，只要添加进来就根据 ordinal 来或运算就可以了。

size 方法就是看指定 long 值的二进制补码表示形式中的 1 位的数量。
